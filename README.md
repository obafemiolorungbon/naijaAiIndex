# Naija AI Index

**The open-source directory of Nigerians building in the AI space.**

A curated, community-driven index of AI projects, tools, and research built by Nigerians. Contributions are welcomed through Pull Requests.

[Browse the Directory](#directory) · [Add Your Project](CONTRIBUTING.md) · [Website](https://naija-ai-index.github.io)

---

## Mission

To create the most comprehensive, community-maintained directory of AI innovation by Nigerians — open, auditable, and accessible to everyone.

## How to Add a Project

1. **Fork** this repository
2. **Create** a YAML file in `data/projects/` named `your-project-slug.yaml`
3. **Fill in** the required fields (see [CONTRIBUTING.md](CONTRIBUTING.md) for the template)
4. **Validate** your entry: `pnpm validate:data`
5. **Update** generated files: `pnpm generate:readme && pnpm generate:data`
6. **Submit** a Pull Request

## Not in Scope

The following are **not** part of this project's mission:

- No job board or job listings
- No advertisements or sponsored placements
- No web scraping or automated data collection
- No closed-source or proprietary-only submissions
- No paywalled lists or premium tiers

This index is and will remain **free, open, and community-driven**.
