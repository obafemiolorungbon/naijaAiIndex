# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email **security@naija-ai-index.dev** (placeholder) with:

1. A description of the vulnerability
2. Steps to reproduce the issue
3. Potential impact assessment
4. Any suggested fixes (optional)

We will acknowledge your report within 48 hours and provide a detailed response within 7 days.

## Scope

This security policy covers:

- The Naija AI Index website and build system
- The data validation and generation scripts
- The CI/CD pipeline configuration

This policy does **not** cover third-party services or websites linked from project entries in the directory.
